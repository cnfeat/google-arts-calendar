

# 🎨 【Google 艺文日历】

## 🚀 项目简介

**Google 艺文日历** 是一款专为 $\text{Tampermonkey}$ 设计的用户脚本，旨在为 $\text{Google}$ 艺术与文化爱好者提供一个集成且高效的浏览环境。

该脚本通过克服跨域 $\text{iframe}$ 安全限制，将 **$\text{edui123}$ 日历** 以一个可交互、可拖拽的精简浮动窗口形式，完美叠加在 **$\text{Google}$ 艺术与文化** 的页面上。同时，它还会自动化滚动页面，让您第一时间看到每日精选艺术品。

---

## ✨ 核心功能亮点

* **双内容高效整合：** 在一个页面上同时浏览 $\text{Google}$ 艺术与文化的主页内容和 $\text{edui123}$ 日历。
* **内容净化与适配：** 脚本自动移除日历网站的页眉、页脚等冗余信息，并精确适配 $\text{612} \times \text{899}$ 的内容比例。
* **自动定位：** 页面加载后，脚本会瞬间滚动到“今日精选”的标题位置，无需手动查找。
* **灵活的浮窗控制：** 浮窗默认贴合左侧边缘，支持**拖拽移动**、**调整大小**和**一键关闭**。
* **高安全性：** 脚本不收集任何信息，仅在您的本地浏览器环境中运行。

## 🛠️ 安装指南

本脚本需要安装 **$\text{Tampermonkey}$** 或其他兼容的浏览器用户脚本管理器。

### 步骤一：安装 Tampermonkey 扩展

请根据您的浏览器（Chrome、Firefox、Edge 等）访问官方应用商店，安装 **$\text{Tampermonkey}$** 扩展程序。

### 步骤二：安装脚本

1.  复制以下链接（指向脚本的 $\text{RAW}$ 文件，请在 $\text{GitHub}$ 仓库创建后替换为您的实际链接）：

    `[您的仓库URL/Google_艺文日历.user.js]`

2.  在浏览器中打开该链接，**$\text{Tampermonkey}$** 将自动识别并弹出安装界面。

3.  点击 **【安装】** 按钮。

### 步骤三：验证

1.  打开 $\text{Google}$ 艺术与文化页面：$\text{[https://artsandculture.google.com/](https://artsandculture.google.com/)}$
2.  页面将自动滚动，“**今日精选**”标题将出现在视口内。
3.  左上角将出现一个可拖拽的 **【edui123 日历】** 浮动窗口。

## ⚙️ 技术原理简述

本项目巧妙地在一个文件中实现了对两个不同域名（$\text{Google}$ 艺术与文化和 $\text{edui123}$ 日历）的操作：

1.  **跨域突破：** 利用 $\text{Tampermonkey}$ 的 **$\text{@match}$** 和 **$\text{@grant\ GM\_addStyle}$** 权限，实现了**“单脚本双域执行”**。当 $\text{iframe}$ 加载日历页面时，脚本的另一部分代码会瞬间激活，执行 $\text{CSS}$ 净化，从而绕开了 $\text{iframe}$ 的安全限制。
2.  **滚动优化：** 使用 $\text{JavaScript}$ 的 `scrollIntoView({ behavior: 'instant', block: 'start' })` 方法，实现了在 $\text{Google}$ 艺术与文化这种动态加载内容 $\text{(SPA)}$ 页面上的 **瞬间、精准定位**。
